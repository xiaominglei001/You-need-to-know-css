使用开发者工具调试 CSS

这是一个交互式教程，将引导您学习如何使用浏览器的开发者工具来检查和调试CSS样式。

点击我

操作指南

  1. 打开开发者工具：在“点击我”按钮上右键单击，然后选择“检查”或“审查元素”。这将在您的浏览器中打开开发者工具。
  2. 定位元素：在开发者工具的“元素”面板中，您应该会看到 <button id="myButton" class="pretty-button"> 这行代码高亮显示。
  3. 查看和修改样式：在“样式”面板中，找到 .pretty-button 类的样式规则。您可以看到我们预设的样式，如 background-color 和 border-radius。
  4. 实时编辑：尝试直接在“样式”面板中单击颜色代码（例如 #3498db）并选择一个新颜色。您会看到按钮的颜色立即发生变化！
  5. 禁用/启用样式：每个CSS属性旁边都有一个复选框。尝试取消勾选 border-radius 旁边的复选框，看看按钮的圆角会发生什么变化。
  6. 添加新样式：在 .pretty-button 规则的任何位置单击，然后输入一个新的CSS属性，例如 box-shadow: 2px 2px 5px rgba(0,0,0,0.3);，然后按Enter键。看看按钮是否出现了阴影效果。
  7. 探索伪类：在“样式”面板中，您可能会看到一个 :hover 选项。点击它可以强制元素处于悬停状态，这样您就可以方便地调试 .pretty-button:hover 的样式了。

提示： 您在开发者工具中所做的所有更改都是临时的。刷新页面后，所有内容都将恢复到原始状态。这使得开发者工具成为一个可以安全进行实验和调试的绝佳环境。
